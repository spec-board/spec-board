## Quick Start

### Option 1: Docker (Recommended)

```bash
docker compose up -d --build
```

Open http://localhost:3000

### Option 2: Local Development

```bash
pnpm install
cp .env.example .env          # Add DATABASE_URL
pnpm prisma migrate dev
pnpm dev
```
